# CTXBench Lattes

CTXBench Lattes is a benchmark instance for evaluating context access and provisioning strategies in LLM-based systems using Brazilian Lattes curricula.

This repository contains:

- benchmark questions;
- instance-specific parameters;
- dataset metadata;
- experiment configurations;
- scripts for downloading and verifying dataset artifacts;
- a small example dataset for testing.

The CTXBench CLI is maintained in [`ctxbench/ctxbench-cli`](https://github.com/ctxbench/ctxbench-cli).
